# React Native News App (Test Task)

This project implements the test task:

- React Native CLI (no Expo)
- TypeScript
- Functional components + hooks
- State management via Zustand
- React Navigation 6 with bottom modal sheet
- News list with FlatList
- Search by title and text
- Create news screen with disabled button until all fields are filled
- API used: https://yourtestapi.com
- SVG icons via `react-native-svg`
- Basic offline support: previously loaded news are cached via AsyncStorage

## Requirements

- Node.js
- Yarn or npm
- Android Studio and/or Xcode configured for React Native CLI

## Installation

```bash
yarn install
# or
npm install
```
